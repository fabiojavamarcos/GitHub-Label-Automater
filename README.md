### How to use
This script requires a flag which determines the type of HTTP request method used, e.g. "-u", standing for "update", uses POST, or "-r", standing for "replace", uses PUT; a text file with properly formatted label inputs (see "inputs" directory); and a text file with properly formated user authentification information.
